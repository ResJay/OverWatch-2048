# OverWatch-2048
2048小游戏
#### 原理
 ```
*根据计算出来的数保存的数组然后再渲染界面

*判断用户滑动方向=>根据用户滑动方向进行计算=>用户每次操作都能获得一个数组=>清空整个页面=>根据数字渲染页面

*得分系统=>用户每操作一次根据数据获得 分数

*用户操作后得到的数据=>用函数过滤后数据:[0,0,1,0,2,0,3,4,4,0,5,0,6,0,7,4,8,0,9,2,10,0,11,0,12,0,13,0,14,0,15,0]
 ```
